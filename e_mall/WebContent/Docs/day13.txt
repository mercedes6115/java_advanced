Script Tag

선언문 : <%!	%> 자바변수(전역 변수)나 매서드 등을 정의 , 		각행은 세미콜론(;)으로 끝나야 함 
표현문 : <%@=	%> 변수 계산식, 매서드 등의 결과를 문자열로 출력,  	같은 조건 
스크립틀릿 : <%	%> 자바 코드를 작성하는데 사용된다(가장 많이 사용)

선언문 : 변수뿐 아니라 매서드도 선언할 수있다
스크릿틀릿: 변수선언만 할수있다

클라이언트 사이드는 결과값만 보여준다 


action tag : 서버나 클라이언트에게 어떤 행동을 하도록 지시하는 태그 
jsp페이지에서 페이지와 페이지 사이를 제어하거나 다른 페이지의 샐행결과 내용을 현재 페이지에 포함
JavaBeans 등의 다양한 기능을 제공 

first.jsp 실행 결과를 보면 처음에 요청한 url에 first.jsp로 되있지만 first.jsp의 내용은 없고
second.jsp의 내용만 있다 


액션 태그는 스크립트 태그와 마찬가지로 jsp에서 자바 코드를 삽입할수 있는 태그이다
액션 태그는 스크립트 태그를 이용하여 jsp 페이지에 직접 자바코드를 작성하는 것을 피하기 위해서 사용된다 
jsp페이지를 작성할떄 자바 코드의 삽입을 최소화 하여 유지보수를 효율적으로 하는 것이 목적이다
